# pro-crawler

## Introduction
This is a learning project for building a web crawler.

## Features
- Basic crawling functionality
- Data extraction

## Tech Stack
- Python
- Requests
- BeautifulSoup

## Usage
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the crawler: `python crawler.py`
